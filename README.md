# Simple CSS Grid Framework with Compass

Simple 12 columns grid framework powered by Compass

## Example Code
```css
<div class="container">
  <div class="row">
    <div class="col6"> Column 6 </div>
    <div class="col6"> Column 6 </div>
  </div>
</div>
```
